# Proyecto-linux-grupo4

## Descripcion
Proyecto de adminstracion de servidor linux con Docker, automatizacion y control de versiones.

## estructura del proyecto
```
/proyecto/
├── datos/          # Archivos de configuración
├── web/            # Archivos del sitio web
├── scripts/        # Scripts de automatización
└── capturas/       # Evidencias del proyecto
```

## Pasos de Instalación

### 1. Configuración del Sistema
- Nombre del host: servidor-grupo6
- Usuarios creados: adminsys, técnico, visitante
- Grupos: soporte, web

### 2. Estructura de Directorios
- Directorios creados en /proyecto/
- Permisos de grupo configurados

### 3. Script de Monitoreo
- Script reporte_sistema.sh configurado
- Tarea cron ejecutándose cada 30 minutos

### 4. Docker
- Docker instalado y configurado
- Usuarios agregados al grupo docker

### 5. Servidor Web
- Contenedor Nginx funcionando
- Sitio web accesible en puerto 8080
